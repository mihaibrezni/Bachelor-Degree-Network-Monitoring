# Bachelor Thesis: Network Monitoring

Hello guys,

This is my final Bachelor thesis which I wanted to share with you. The thesis is split into **2 main areas**:

1. **Replicate** part of the current CERN network monitoring solution on the QKD network of UPB.
2. **Interpret** the metrics collected from CERN infrastructure and process them to get relevant information.
